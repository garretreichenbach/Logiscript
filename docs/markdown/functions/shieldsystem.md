# ShieldSystem API

`ShieldSystem` exposes aggregate and per-bubble shield data.

## Reference

- `isShielded()`
- `getCurrent()`
- `getCapacity()`
- `getRegen()`
- `getAllShields()`
Returns `Shield[]`.

- `getActiveShields()`
Returns active `Shield[]`.

- `isShieldActive(index)`
Checks by shield index.
