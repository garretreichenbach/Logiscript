# Vector API

`LuaVec3i` (Vec3i wrapper) is used for block/sector/system positions.

## Reference

- `getX()` / `getY()` / `getZ()`
- `setX(x)` / `setY(y)` / `setZ(z)`
- `add(vec)`
- `sub(vec)`
- `mul(vec)`
- `div(vec)`
- `scale(number)`
- `absolute()`
- `negate()`
- `size()`

## Notes

- Most math mutates and returns the same vector instance.
