# Chamber API

`Chamber` wraps a reactor chamber/specialization node.

## Reference

- `getName()`
- `getBlockInfo()`
- `getReactor()`
- `specify(name)`
Attempts to switch to a valid specification by block/chamber name.

- `getValidSpecifications()`
- `isUsable()`
- `getUsable()`
Returns `UsableChamber` when this chamber supports active use.
